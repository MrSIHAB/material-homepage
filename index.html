About

Material_homepage is a Nee Tab extension to ensure the flexibility of choosing search engine. This is a open source project of Mine. There is no risk at all in this extension. The source code is available in github. Here's some minimal overview of this extension. This extension is minimal designed with no destruction. Freshly designed for focus mode. I've added all apps section where maximum popular apps is available. More popular apps and dev mode will be available soon. Custom shortcut is not available right now. But I've planned to bring that while we get strong user support. I've Added quick mail section to easily check your gmail, yahoo mail, outlook and hotmail.
